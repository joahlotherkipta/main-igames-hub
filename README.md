# main-igames-hub

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。内容以静态页面形式存放，不针对任何特定平台或站点。适合作为网页素材、小型工具页面或学习样例的集合地。

## 目录说明

```
.
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 公共资源文件（图片、样式、脚本等）
├── docs/           # 说明文档或附加资料
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件名对应一个独立页面。
- `assets/` 中的资源文件供页面引用，保持结构清晰。
- `docs/` 可存放使用说明、版本记录等。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，可单独打开使用。
- 部分页面可能引用 `assets/` 下的资源，确保路径正确即可。
- 归档内容可能不定期更新或整理。

## 维护说明

- 欢迎提交 Issue 反馈问题或建议。
- 如需添加新页面，请遵循现有目录结构。
- 提交 Pull Request 前请确保页面功能正常且无冗余文件。
- 本仓库主要用于个人归档与分享，不保证长期更新频率。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件。
3. 或使用任意静态服务器托管整个目录（如 Python 的 `http.server`）。
